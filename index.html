About

Aplicación para realizar seeding automático del lol. El objetivo es realizar una aplicación en la que puedas crear distintos equipos, almacenarlos en localstorage, y automáticamente realice el seeding según sus rangos.

Esta aplicación no se encuentra avalada, respaldada por; ni tiene ningún tipo de relación con Riot Games aparte de la utilización de su API pública.

Para esto, utilizo la API oficial de Riot Games, AngularJS y jQuery entre otras tecnologías. Marina es una pesada, espero que esto haga que se estrese un poco menos.

This application is in no way supported, endorsed or related by Riot Games; besides the use of their public API.
